# Weight Converter
A simple weight converter project in which the user enters the weight in kilograms and it gets converted to 
grams, pounds and ounces.

# Author
- [@palaksinghania05](https://github.com/palaksinghania05)

# Stack Used
- HTML, CSS, JavaScript
